    54вЖТ## рЯУЦ Overview
    55вЖТ
    56вЖТ**DAKKHO Student App** is a Netflix-like streaming application for polytechnic students in Bangladesh. Built with Next.js 16 as a static SPA, it provides course browsing, video playback, progress tracking, bookmarks, achievements, social features, 20+ polytechnic department pages, and semester pages вАФ all communicating with a shared Cloudflare Worker API.
    57вЖТ
    58вЖТThe app runs entirely on **Cloudflare Pages** as a static export (`output: "export"`) with client-side routing powered by Zustand stores and the History API.

   101вЖТ**Key architectural decisions:**
   102вЖТ
   103вЖТ- рЯМН **Edge-first** вАФ Student App is a static SPA served from Cloudflare Pages CDN; all data flows through the shared Worker API
   104вЖТ- рЯЧДпЄП **Shared D1 (SQLite) database** вАФ Student App, Admin Panel, and Instructor App all read/write to the same D1 database via the Worker API
   105вЖТ- рЯОѓ **Static SPA deployment** вАФ `output: "export"` for zero-cost static hosting on Cloudflare Pages, with client-side routing via Zustand + History API
   106вЖТ- вЪ° **KV-cached server config** вАФ UI configuration (feature flags, navigation, content protection) is fetched from `GET /api/config` and cached in a Zustand store
   107вЖТ- рЯУ¶ **R2 media delivery** вАФ Videos, thumbnails, avatars, and resources are served from R2 buckets through the Worker API with 7-day browser cache headers

   236вЖТ| 9 | `useServerConfigStore` | Server-driven UI config вАФ feature flags, home sections, sidebar sections, bottom nav tabs, top bar elements, card style | Fetched from `/api/config` |
